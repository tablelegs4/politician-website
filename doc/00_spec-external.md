# Material Specification
## 素材・画像・メディア仕様

本ドキュメントは、政治家HPテンプレを制作・運用するために必要な  
**素材の種類・チェックリスト・技術仕様**を定義する。

本ドキュメントは実務仕様であり、思想・構造・コンテンツ方針の説明は行わない。  
それらは `design-principles.md` および `spec.md` を前提とする。

素材仕様に関する最終的な判断は、本ドキュメントを唯一の正とする。

---

## 1. 基本方針

- 必要な素材を定義し手戻りを防ぐことを目的とする
- 素材の定義を限りなく明確にする
- GitHub Pages 等の静的サイト運用を前提とする
- 素材を作成する過程で演説や討論会の資料になりうる工夫をする
