# 政治家公式HP テンプレート概要

クライアント向けの説明資料です。

---

## 1. サイトの目的

このサイトは、閲覧者が政治家の活動・方針を理解し、信頼し、次の行動に移ることを目的としています。

**主な行動:**

- 相談・要望を送る
- 活動レポートを受け取る
- SNSをフォローする

※ 直接的な投票依頼は行いません（平時運用）

---

## 2. ページ構成

### 必須ページ

| ページ | 内容 |
| --- | --- |
| トップ | メインビジュアル、重点政策、伝えたいこと、CTA、FAQ |
| 政策一覧 | 全政策の一覧 |
| 政策詳細 | 各政策の詳細（課題→提案→進め方→目標→コスト→Q&A） |
| プロフィール | 経歴、政治信条、基本情報 |
| お問い合わせ | 相談・要望フォームへの導線 |
| プライバシーポリシー | 個人情報の取り扱いについて |

### 任意ページ

- 議会資料・一般質問一覧
- 掲載実績
- 実績
- 参加・協力

---

## 3. 必要な素材・データ

### 画像素材

| 素材 | 用途 | 推奨サイズ |
| --- | --- | --- |
| メインビジュアル | ファーストビュー背景 | 1920x1080px以上 |
| プロフィール写真 | プロフィールページ、サイドバー | 400x400px以上 |
| ロゴ画像 | ヘッダー | 党名・肩書き・名前を含む |
| OGP画像 | SNSシェア時のサムネイル | 1200x630px |

### テキストデータ

| 項目 | 内容 | 文字数目安 |
| --- | --- | --- |
| キャッチコピー | 立場が一瞬でわかるメッセージ | 15〜25字 |
| サブコピー | 誰のために／何を改善するのか | 2行程度 |
| 重点政策（3つ） | 見出し + 説明 | 各100〜140字 |
| 伝えたいこと（2本） | 問い + 回答形式 | 各300字以内 |
| プロフィール短文 | TOPページ用 | 200字程度 |
| FAQ（3件） | 質問 + 回答 | 各100〜200字 |

### プロフィール情報

**必須項目:**

- 氏名
- 二つ名（キャッチフレーズ）
- 生年（年のみ可）
- 座右の銘
- 尊敬する政治家
- 小ネタ（1〜2個）
- 経歴
- 政治信条

**任意項目:**

- 出身地
- 現住所
- 家族構成
- 選挙区情報
- 実績

### 政策データ

各政策に以下の情報が必要です:

1. 政策タイトル
2. 課題（現状の問題点）
3. 提案（何をするか）
4. 進め方（短期/中期/長期）
5. 目標
6. コスト
7. Q&A（想定される質問と回答）
8. 参考資料（外部リンク可）

### 連絡先情報

- 事務所住所（Google Mapリンク用）
- メールアドレス（またはお問い合わせフォームURL）
- SNSアカウント（X、Facebook、Instagram、LINE等）

## 6. 運用上の注意点

### 平時運用のルール

- 直接的な投票依頼は行わない
- 対立を煽る断定表現は避ける
- 法解釈・医療に関する断定は避ける

### 推奨表現

- 「〜を実現します」「〜を提案します」
- 「相談を受け付けています」
- 「活動報告」「実績」「提案内容」

### 更新頻度の目安

| コンテンツ | 推奨頻度 |
| --- | --- |
| 活動報告 | 週1〜2回 |
| 政策ページ | 必要に応じて |
| プロフィール | 役職変更時など |
